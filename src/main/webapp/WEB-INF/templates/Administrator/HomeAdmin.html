Bienvenue dans l'espace Administrateur de PRONOMASTER !

De nombreuses fonctionnalités sont ici disponnibles afin d'assurer au mieux le bon déroulement des parties des joueurs. Parmi lesquelles nous retrouvons...

Ajouter des matchs

Rendez-vous sur l'onglet "Ajouter un match" pour compléter la liste des matchs sur lesquelles les joueurs pourront immédiatement parier

Inscrire les résultats

Une fois un match terminé, il vous sera possible de renseigner son résultat en vous rendant dans l'onglet "Entrer des scores". Une fois le score inscrit, les joueurs verront automatiquement leur score évoluer (ou non) en conséquence.

Gérer les membres

L'onglet "Membres" vous permettra d'effectuer différentes opérations sur les membres de Pronomaster :

  • Visualiser l'ensemble des membres, aussi bien administrateur que joueurs, avec un code couleur vous permettant de les différencier
  • Si par exemple un membre à un comportement indésirable, ou si il est innactif pendant trop longtemps, il vous sera possible de le banir en cliquant sur "bannir"
  • Si vous désirez changer le statut d'un utilisateur, nommer quelqu'un administrateur ou inversement, la fonctionnalité est également disponnible
